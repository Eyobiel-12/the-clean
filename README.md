# The Clean – Premium Website

A premium website for The Clean, a professional cleaning services company. Built with React, Framer Motion, and GSAP for smooth, high-end animations.

## Tech Stack

- **React 18** + **Vite**
- **Framer Motion** – Component animations, hover effects, page transitions
- **GSAP** – Scroll-triggered animations, timeline sequences
- **Tailwind CSS** – Styling
- **Instrument Serif** & **Outfit** – Typography

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```

## Features

- Hero section with GSAP text reveal
- Scroll-triggered section animations
- Responsive navigation with mobile menu
- Image gallery with hover effects
- Contact form
- Dutch content for The Clean services
